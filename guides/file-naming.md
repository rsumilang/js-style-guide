# File Naming

Public JavaScript classes should be in a single .js file and match the name
of the class.

[JavaScript](../js/file-naming.js)

*Right:*

```js
// MyClass.js
var MyClass = function MyClass () {
  // magic.
};
```

*Wrong:*

```js
// my-class.js
var MyClass = function MyClass () {
};
```
